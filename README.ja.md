> **新機能!!**
> `sandbox`リポジトリを選択した場合、`sandbox`内のディレクトリで再びファジーファインダーを開きます。

## 🚀 使い方

ghqに依存しています。[ghq](https://github.com/x-motemen/ghq)

```
go build
./recker
```

## ⬇️ Install

任意のPATH上にバイナリを配置してください。

## ⛏️ 開発

```sh
go run .
```
